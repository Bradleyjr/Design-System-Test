# Changelog

All notable changes to this design system will be documented in this file.

## [1.1.0] - 2024 (Extended Release)

### Added - New Atom Components (6)
- **Checkbox** - Customizable checkbox with label support and sizes
- **Radio** - Radio button component with label and error states
- **Textarea** - Multi-line text input with resize options
- **Spinner** - Loading spinner with multiple sizes and colors
- **Avatar** - User avatar component with image, initials, and status indicator support
- **Divider** - Horizontal/vertical divider with optional label

### Added - New Molecule Components (5)
- **Select** - Dropdown select component with option groups
- **Tooltip** - Contextual tooltip with multiple positioning options
- **Pagination** - Page navigation with ellipsis and configurable sibling count
- **Breadcrumbs** - Navigation breadcrumb trail with collapsing support
- **ProgressBar** - Progress indicator with variants, labels, and animation

### Added - New Organism Components (2)
- **Modal** - Accessible modal dialog with multiple sizes, focus trap, and keyboard navigation
- **Table** - Data table with sorting, variants (striped, bordered), and loading states

### Added - New Design Tokens (2)
- **Animations** - Animation durations, easing functions, and common transitions
- **Z-Index** - Layering system for consistent stacking contexts

### Improvements
- Enhanced type safety across all components
- Improved accessibility with ARIA labels and keyboard navigation
- Better responsive behavior for mobile devices
- Added comprehensive prop types for all new components
