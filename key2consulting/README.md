# Key2 Consulting Website

A modern, professional website for Key2 Consulting - a premier data analytics consultancy specializing in Microsoft technologies.

## Overview

This website showcases Key2 Consulting's expertise in data warehousing, business intelligence, data integration, and custom application development. Built entirely with our custom design system, it demonstrates how atomic design principles create cohesive, scalable web applications.

## About Key2 Consulting

- **Industry**: Data Analytics & Business Intelligence Consulting
- **Specialization**: Microsoft Azure, Power BI, SQL Server
- **Experience**: 20+ years in business
- **Team**: 40+ expert consultants
- **Partnership**: Microsoft Gold-Certified Partner

   - Final call-to-action

### Design System Components Used

#### Atoms
- Button (multiple variants: primary, secondary, outline)
- Heading (levels 1-4)
- Text (various sizes and colors)
- Badge (for highlighting key information)
- Input, Textarea, Label (form elements)
- Avatar (for team members)

#### Molecules
- Card (with CardHeader, CardBody, CardFooter)
- FormField (combining label, input, validation)
- Badge (status indicators)

#### Organisms
- Header (with navigation and branding)
- Footer (with multiple sections and links)
- Hero (large banner sections)
- ContactForm (complete form with validation)

#### Templates
- MainLayout (standard page layout with header and footer)
